# Knowledge Base Rights and Attribution

This distribution is source-grounded in the public Rob Burbea digital garden, but it does not own or vendor that corpus.

Source repository:

```text
https://github.com/fschuhi/digital-garden-rob-burbea-publish
```

The source repository describes itself as a collection of Obsidian Markdown files published as "The Garden of the Soul: Rob Burbea's Teachings." Its README states that the Hermes Amara Foundation holds the rights to Rob's talks.

## Distribution Policy

- Do not commit the cloned garden into this repository.
- Clone it locally with `scripts/clone_or_update_garden.py`.
- Build local indexes under `knowledge/indexes/`.
- Cite source paths when using the corpus.
- Summarize rather than copying long passages.
- Keep direct quotations short and source-specific.
- Do not imply this profile is affiliated with or endorsed by the Hermes Amara Foundation.

## Runtime Dependency

The expected local path is:

```text
knowledge/rob-burbea-garden/
```

This path is ignored by git.
