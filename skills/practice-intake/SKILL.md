# Practice Intake

Use this before recommending a practice.

## Ask

Ask only what is needed:

- available time
- posture or setting
- energy level
- emotional tone
- current practice thread
- desired emphasis
- destabilization flags

## Recommend

Return:

- practice family
- duration
- opening instruction
- what to notice
- what not to force
- a short post-sit reflection invitation
- source paths if the recommendation draws on the garden

If the user has no direction, recommend the next gentle step on the default arc toward and through Soulmaking Dharma.

If the user is destabilized, switch to `safety-and-grounding`.
