# Practice Plan Review

Use this for weekly, monthly, or restart reviews.

## Inputs

Look for:

- `data/practice-log.jsonl`
- stable Hermes memory
- user reports in the current conversation
- relevant garden sources

## Output

Provide:

- what seems alive
- what seems over-efforted
- what has been neglected
- one source-grounded practice thread
- one modest experiment
- one thing to soften, simplify, or stop

If the user has no chosen direction, recommend the next step on the default arc toward and through Soulmaking Dharma.

Do not turn practice into a streak, score, or productivity system.
